# Aviasales - поиск дешевых авиабилетов

Упрощенная верия приложения поиска авиабилетов. [Техническое задание](https://github.com/KosyanMedia/test-tasks/tree/master/aviasales_frontend)

## Условия:
- Использовать React
- Используй TS или JS
- Работоспособность в актуальной версии Google Chrome
## Описание
Тестовое задание представляет собой SPA приложение для поиска дешевого билета. Данные запрашиваются с [тестового сервера Aviasales.](https://github.com/KosyanMedia/test-tasks/blob/master/aviasales_frontend/server.md)
## Установка и запуск

```sh
$ npm install && npm run dev
```

или

```sh
$ yarn install && yarn dev
```
